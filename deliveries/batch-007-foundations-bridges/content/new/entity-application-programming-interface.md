# Application Programming Interface

## Contexto y frontera

Una API especifica puntos de interacción, operaciones, datos y reglas mediante los cuales un consumidor usa capacidades de software sin depender de todos sus detalles internos. No equivale a una marca, herramienta o implementación única.
