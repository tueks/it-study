# Seguridad

## Propósito educativo

Muestra cómo una disciplina con hogar propio se aplica en todo IT sin convertir cada aplicación en otra entrada.

## Frontera

No equivale solo a operaciones de seguridad ni absorbe privacidad, riesgo o cumplimiento como sinónimos.
