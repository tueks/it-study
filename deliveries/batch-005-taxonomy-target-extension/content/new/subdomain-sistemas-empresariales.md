# Sistemas empresariales

## Propósito educativo

Contextualizar sistemas como ERP o CRM por las capacidades que integran y no como listas de productos comerciales.

## Frontera

Cualquier aplicación interna queda fuera de este subdominio; la pertenencia exige que el sistema coordina procesos o información entre varias capacidades organizacionales.
