# Accesibilidad y diseño inclusivo

## Propósito educativo

Tratar accesibilidad como requisito y disciplina transversal desde el inicio, no como corrección visual tardía.

## Frontera

Cumplimiento mecánico sin experiencia queda fuera de este subdominio; la pertenencia exige que el tema reduce barreras de acceso o uso para personas diversas.
