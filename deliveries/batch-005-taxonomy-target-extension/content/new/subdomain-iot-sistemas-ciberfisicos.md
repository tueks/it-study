# IoT y sistemas ciberfísicos

## Propósito educativo

Explicar la interacción entre dispositivos, redes, software y entorno físico sin reducirla a sensores o productos conectados.

## Frontera

Todo sistema embebido queda fuera de este subdominio; la pertenencia exige que existe interacción esencial entre cómputo y un proceso físico.
