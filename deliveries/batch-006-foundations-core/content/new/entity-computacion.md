# Computación

## Contexto

La computación abarca ideas, métodos y sistemas para expresar y ejecutar procesos sobre información. Incluye fundamentos teóricos y práctica profesional, pero no se reduce a una herramienta, lenguaje o dispositivo.

## Distinción clave

Programar es una actividad dentro de la computación. Un algoritmo expresa un procedimiento; un programa lo representa para que un sistema pueda ejecutarlo.
