# Lógica booleana

## Contexto

Las operaciones booleanas permiten construir condiciones simples y combinadas. Aparecen en programas, circuitos, filtros y reglas, aunque cada aplicación añade su propio contexto.

## Distinción clave

La lógica booleana es un modelo formal; una condición de negocio es una regla contextual que puede expresarse parcialmente con ese modelo.
