# DevOps e ingeniería de plataformas

## Propósito educativo

Permite distinguir un enfoque socio-técnico, una capacidad de plataforma y las herramientas que pueden apoyarlos.

## Frontera

No presenta DevOps como puesto, equipo obligatorio, producto o secuencia universal de herramientas.
