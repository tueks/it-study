# Middleware e integración

## Propósito educativo

Ayuda a reconocer la capa de mediación que aparece cuando dos sistemas no pueden cooperar mediante una conexión directa simple.

## Frontera

No convierte una marca, bus o plataforma concreta en sinónimo de integración empresarial.
