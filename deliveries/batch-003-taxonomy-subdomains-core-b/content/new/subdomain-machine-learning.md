# Machine learning

## Propósito educativo

Permite reconocer qué cambia cuando un sistema depende de modelos aprendidos y por qué entrenamiento, evaluación e inferencia son actividades distintas.

## Frontera

No equipara toda inteligencia artificial con machine learning ni todo análisis estadístico con un sistema de IA.
