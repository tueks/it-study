# Arquitectura y gestión de datos

## Propósito educativo

Permite reconocer que almacenar datos no basta para compartir su significado, calidad, responsabilidad y uso permitido.

## Frontera

No convierte todo trabajo con datos en gobierno ni sustituye la ingeniería que implementa sus flujos.
