# Operaciones de seguridad

## Propósito educativo

Ubica detección y respuesta como capacidades operativas continuas, distintas de diseñar controles o gobernar el riesgo completo.

## Frontera

No reduce seguridad a un centro de operaciones ni asume una estructura organizacional única.
