# Arquitectura empresarial

## Propósito educativo

Permite reconocer una escala de arquitectura orientada a coherencia organizacional, distinta del diseño interno de un sistema particular.

## Frontera

No presupone un framework, conjunto de capas o modelo operativo universal.
