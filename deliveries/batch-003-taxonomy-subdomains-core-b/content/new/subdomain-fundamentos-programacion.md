# Fundamentos de programación

## Propósito educativo

Sitúa el vocabulario necesario para reconocer cómo un problema se expresa como datos, operaciones y flujo de control.

## Frontera

No enseña a programar paso a paso ni representa por sí solo el desarrollo profesional de software.
