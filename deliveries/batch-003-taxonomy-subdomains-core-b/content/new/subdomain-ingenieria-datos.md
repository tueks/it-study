# Ingeniería de datos

## Propósito educativo

Distingue la construcción de flujos de datos de su gobierno, almacenamiento aislado y análisis final.

## Frontera

No implica una arquitectura de plataforma única ni convierte cualquier transformación de datos en un pipeline independiente.
