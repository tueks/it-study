# Sistema operativo

## Contexto y frontera

Un sistema operativo coordina procesador, memoria, dispositivos, almacenamiento y ejecución para proporcionar un entorno controlado a aplicaciones. Su uso concreto depende del entorno y no elimina la necesidad de validar supuestos.
