# Proceso de sistema operativo

## Contexto y frontera

Un proceso es una unidad de ejecución aislable que incluye código, datos, estado y recursos gestionados por un sistema operativo. Su uso concreto depende del entorno y no elimina la necesidad de validar supuestos.
