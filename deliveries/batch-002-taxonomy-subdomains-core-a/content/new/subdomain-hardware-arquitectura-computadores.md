# Hardware y arquitectura de computadores

## Propósito educativo

Permite ubicar procesador, memoria, almacenamiento y dispositivos dentro de una organización coherente del sistema.

## Frontera

No sustituye electrónica especializada ni la provisión y operación de infraestructura.
