# APIs e interoperabilidad

## Propósito educativo

Separa la existencia de una conexión del acuerdo necesario para que sistemas cooperen de forma útil.

## Frontera

Las redes transportan; backend implementa; este subdominio se centra en contratos e interoperabilidad entre sistemas.
