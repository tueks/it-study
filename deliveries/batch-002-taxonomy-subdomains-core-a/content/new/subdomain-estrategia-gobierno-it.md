# Estrategia y gobierno de IT

## Propósito educativo

Ubica dirección, derechos de decisión y supervisión para comprender cómo IT se conecta con objetivos y riesgo.

## Frontera

No sustituye gestión, proyectos, arquitectura empresarial ni gobiernos especializados de seguridad, datos o IA.
