# Diseño y arquitectura de software

## Propósito educativo

Permite reconocer decisiones estructurales y sus compromisos más allá de producir código funcional.

## Frontera

La arquitectura empresarial conserva hogar en estrategia y gobierno; el diseño visual pertenece a producto y experiencia.
