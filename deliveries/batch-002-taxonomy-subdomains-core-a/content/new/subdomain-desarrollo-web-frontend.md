# Desarrollo web frontend

## Propósito educativo

Separa la implementación de una interfaz web de las decisiones de experiencia, producto y servicios del servidor.

## Frontera

La presencia de una interfaz no traslada aquí investigación UX, accesibilidad como disciplina ni lógica central del negocio.
