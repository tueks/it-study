# Bases de datos

## Propósito educativo

Ubica modelos y sistemas de persistencia dentro de una visión más amplia del ciclo del dato.

## Frontera

No incluye automáticamente pipelines, gobierno, BI o IA, aunque esas capacidades consuman bases de datos.
