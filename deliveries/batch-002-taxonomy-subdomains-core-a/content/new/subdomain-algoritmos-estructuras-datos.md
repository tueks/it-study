# Algoritmos y estructuras de datos

## Propósito educativo

Conecta procedimientos, representación y costo para construir un vocabulario común antes de estudiar implementaciones.

## Frontera

No clasifica tecnologías de almacenamiento ni fragmentos de código; conserva el nivel conceptual y transferible.
