# Ingeniería de requisitos

## Propósito educativo

Muestra que los requisitos se descubren, negocian, verifican y cambian durante el ciclo de vida.

## Frontera

No sustituye la gestión de producto ni la construcción técnica; conecta ambas mediante necesidades trazables.
